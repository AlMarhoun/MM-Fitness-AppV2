# MM Fitness App Website Deploy Files

Upload all files in this folder to the GitHub Pages branch or root configured for the site.

Required files:

- `index.html`
- `manifest.json`
- `sw.js`
- `src/`
- `assets/`
- `.nojekyll`

Do not remove `.nojekyll`.

After upload, open the site URL with:

`?splash=1&v=11`

Example:

`https://YOUR-USERNAME.github.io/YOUR-REPO/?splash=1&v=11`

If an old PWA version appears on mobile, delete the old home-screen app icon and add it again after opening the new URL.
